How to set up your own personal website?

如何制作个人网站?


第一步：下载相关工具软件（Sublime Text等）、Bootstrap响应式文件

第二步：下载模板（或自行创建），利用相关工具软件进行代码的修改与编写

第三步：（可先用Chrome浏览器等查看网页是否合格）修改完毕，生成html文件

第四步：将已完成的html文件连同css、js等相关文件一并封装打包

第五步：（以新浪SAE部署为例）注册新浪云SAE账号，选择合适语言，将打包好的网页文件上传部署即可
